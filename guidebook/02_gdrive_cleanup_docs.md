# Google Drive Cleanup Documentation

## Overview
This tool automates the cleanup of old files in your Google Drive.

## Usage
1. Configure credentials in `credentials.json`.
2. Run the script: `python bridge/gdrive_cleanup.py`.
3. Follow the prompts to authorize access.

## Features
- Scans specific folders.
- Deletes files older than a threshold.
- Generates a report of actions taken.
